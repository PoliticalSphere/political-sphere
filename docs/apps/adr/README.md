ADR (Architectural Decision Record) guidelines

Place ADRs in `docs/adr/` with a filename like `0001-descriptive-title.md`.

Required front-matter keys:

- `title` (string)
- `date` (YYYY-MM-DD)
- `status` (proposed|accepted|deprecated)
- `deciders` (array of strings)

Use the template `0000-template-record-decision.md` as a starting point.
